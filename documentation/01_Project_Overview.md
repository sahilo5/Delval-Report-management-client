# Project Overview

## Description
The **Delval Report Management Client** is a frontend application built for a company to manage reports. It is designed to interpret and display complex data for various stakeholders in the assembly and quality assurance process.

## Key Features
- **Role-Based Access Control (RBAC)**: Tailored dashboards for different user roles.
- **Custom UI Library**: A set of reusable, accessible components built from scratch for this project.
- **Modern Tech Stack**: Leveraging the latest React ecosystem tools for performance and developer experience.

## User Roles
The application supports the following roles, each with specific permissions and views:

1. **Assembly Engineer**: Responsible for overseeing the assembly process.
2. **Assembler**: Executes assembly tasks and logs progress.
3. **Tester**: Performs initial testing on assembled units.
4. **QA (Quality Assurance)**: Final verification and quality control.
5. **More**: Additional roles may be added as needed.

## Technology Stack
- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Routing**: [React Router v7](https://reactrouter.com/)
